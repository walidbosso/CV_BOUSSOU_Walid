# Personal CV - BOUSSOU WALID

</div>





This repository contains an HTML+CSS personal CV created by BOUSSOU WALID from scratch.

## Live Demo

You can view the live demo of this CV hosted on [w3spaces](https://walidbosso.w3spaces.com).

## Alternative Link

An alternative link to the CV is available at [walidbosso.github.io/CV_BOUSSOU_Walid/](https://walidbosso.github.io/CV_BOUSSOU_Walid/).

## Topics Covered

- HTML5
- CSS3
- CSS Animations
- CV Design with HTML and CSS
- Box Shadow
- Text Shadow
- Transformations in CSS
- Responsive Design
